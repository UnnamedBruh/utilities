DOS Message Editor

This HTML5 experiment allows you to change the DOS error message of almost any Windows EXE!

You can change the message from "This program cannot be run in DOS mode." to "Sorry, but you have to run this on Windows!" to nearly any message you want! The only limit is the maximum number of characters, which cannot be exceeded while guaranteeing original behavior. The max. character limit varies depending on the EXE.

This is NOT guaranteed to work safely. This is in a beta stage.
